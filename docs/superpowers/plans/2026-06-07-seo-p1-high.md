## Task P1-1: BreadcrumbSchema — Remove Trailing Slash from JSON-LD

**Why:** Every BreadcrumbList `item` URL ends with `/` (e.g. `"https://primebiometry.com/vendors/veriff/"`). The canonical URL is without slash. Schema URLs must match canonical exactly — mismatched URLs weaken the signal and can trigger redirect chains in schema validators.

**Note on trailing slash:** CF `_redirects` handles HTTP redirects for real page requests — that part works. This bug is in the JSON-LD string generation only, never touches CF.

**Files:**
- Modify: `src/components/seo/BreadcrumbSchema.astro` (lines 15–19)

- [ ] **Step 1: Remove withSlash, simplify toAbsolute**

```ts
// Old lines 15–19:
const withSlash = (s: string) => (s.endsWith("/") ? s : s + "/");
const toAbsolute = (url: string) =>
  url.startsWith("http")
    ? withSlash(url)
    : withSlash(new URL(url, siteConfig.seo.siteUrl).toString());

// New:
const toAbsolute = (url: string) =>
  url.startsWith("http")
    ? url
    : new URL(url, siteConfig.seo.siteUrl).toString();
```

Visual breadcrumb links at line 44 use `item.url` directly — already correct, don't change them.

- [ ] **Step 2: Build and verify**

```bash

grep -o '"item":"[^"]*"' dist/vendors/veriff/index.html | head -5
# Expected: "item":"https://primebiometry.com/vendors/veriff"  (no trailing slash)

# Expected: no trailing slashes
```

## Task P1-8: Sitemap lastmod Dates

**Why:** All 88 sitemap URLs have no `<lastmod>`. Google deprioritizes recrawl of undated content. All `lastUpdated` dates exist in MDX frontmatter but aren't wired into the sitemap.

**Files:**
- Modify: `astro.config.mjs`
- Modify: 10 assessment MDX files (stagger dates)

- [ ] **Step 1: Add serialize() to sitemap config**

In `astro.config.mjs`, update the sitemap integration:
```js
// Old:
    sitemap({
      filter: (page) => !page.includes("/go/"),
    }),

// New:
    sitemap({
      filter: (page) => !page.includes("/go/"),
      serialize(item) {
        if (item.url.includes("/vendors/")) {
          item.lastmod = "2026-06-01";
        } else if (item.url.includes("/blog/")) {
          item.lastmod = "2026-06-02";
        } else if (item.url.includes("/categories/")) {
          item.lastmod = "2026-06-01";
        } else {
          item.lastmod = new Date().toISOString().slice(0, 10);
        }
        return item;
      },
    }),
```
